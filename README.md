يهدف هذا المشروع إلى تصميم وتنفيذ نظام تواصل غير لفظي يساعد الأشخاص غير القادرين على الكلام، مثل مرضى الشلل أو اضطرابات النطق، على التعبير عن احتياجاتهم ومشاعرهم من خلال حركات اليد.
يعتمد النظام على تحليل إيماءات اليد وتحويلها إلى رسائل أو أوامر مفهومة، مما يسهم في تحسين التواصل وزيادة الاستقلالية ورفع جودة الحياة لهؤلاء الأشخاص.
تم تنفيذ المشروع وفق الخطوات التالية:
التقاط حركة اليد باستخدام كاميرا في الزمن الحقيقي.
معالجة الفيديو باستخدام لغة Python.
استخدام خوارزميات تحليل الصور والرؤية الحاسوبية لاكتشاف اليد.
تحديد نقاط اليد الأساسية وتحليل اتجاه وحركة الأصابع.
مقارنة الحركات المكتشفة مع مجموعة إيماءات مُعرّفة مسبقًا.
تحويل كل إيماءة إلى كلمة، جملة قصيرة، أو رمز تواصلي يظهر على الشاشة أو يُرسل كنص/تنبيه.
اعتماد شروط زمنية وعتبات حركة لتقليل الأخطاء وزيادة دقة التعرف.
